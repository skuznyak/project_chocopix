# Завдання для Codex — перепозиціонувати сторінку /marshmallow під “крафтове маршмелоу”

## Контекст
Проєкт: ChocoPix  
Сайт: https://chocopix.store

У поточній реалізації сторінки `/marshmallow` використовується формулювання `gourmet marshmallow`.  
Його треба прибрати як основне позиціонування сторінки, тому що для українського пошуку воно слабке й неприродне.

Основне нове позиціонування сторінки:
- **маршмелоу купити**
- **крафтове маршмелоу**
- **маршмелоу для какао**
- **маршмелоу для подарунків**
- **маршмелоу ручної роботи**

---

## Головна ціль
Оновити SEO і текст сторінки `/marshmallow`, щоб:
1. прибрати акцент із `gourmet marshmallow`
2. замінити його на природне українське позиціонування
3. зберегти комерційний стиль сторінки
4. не зробити сторінку переспамленою
5. не змінювати дизайн, верстку або архітектуру

---

## Важливі обмеження
1. Працюй **тільки** зі сторінкою `/marshmallow` і файлами, які прямо впливають на її SEO.
2. **Не змінюй** інші сторінки сайту.
3. **Не змінюй UI / стилі / layout**.
4. **Не додавай довгі SEO-тексти** у стилі статті або енциклопедії.
5. Залиш сторінку **магазинною і комерційною**.
6. Не ламай роутинг, prerender, sitemap, schema або existing SEO architecture.
7. Якщо щось уже реалізовано правильно — не переписуй без причини.

---

## Що саме потрібно змінити

### 1. Замінити позиціонування
Прибери або зведи до мінімуму використання:
- `gourmet marshmallow`

Замінити на природніші формулювання:
- `крафтове маршмелоу`
- `маршмелоу ручної роботи`
- `маршмелоу для какао`
- `маршмелоу для подарунка`
- `маршмелоу купити`

> Важливо: якщо англомовна фраза десь потрібна другорядно для стилю картки чи назви смаку — не роби її головним SEO-акцентом.

---

### 2. Оновити Title
Поточне позиціонування з `gourmet marshmallow` заміни на українське.

Орієнтир для нового title:
**Маршмелоу купити — крафтове маршмелоу для какао та подарунків | ChocoPix**

Перевір:
- щоб title був природним
- щоб не був занадто довгим
- щоб зберігав комерційний інтенд
- щоб головний акцент був на `маршмелоу купити`

---

### 3. Оновити Meta Description
Перепиши meta description так, щоб:
- не було акценту на `gourmet marshmallow`
- був природний український опис
- були сенси: ручна робота / крафтове / для какао / для подарунка / доставка

Орієнтир:
**Крафтове маршмелоу ручної роботи: для какао, подарунків і десертів. Купити маршмелоу з доставкою по Україні від ChocoPix.**

Не копіюй сліпо. Підженяй під реальний контент сторінки.

---

### 4. Оновити H1
Заміни поточний H1 на формулювання без `gourmet marshmallow`.

Орієнтир:
**Маршмелоу купити для какао та подарунків — крафтове виробництво**

Допускається дуже близький варіант, якщо він краще лягає в UI.

---

### 5. Оновити підзаголовки / короткий текст сторінки
Переглянь H2/H3 та короткі SEO/описові блоки сторінки.

Потрібно:
- підсилити формулювання `крафтове маршмелоу`
- зберегти комерційну подачу
- не зробити текст перевантаженим
- не використовувати штучні або дивні SEO-фрази

Орієнтири для сенсів:
- крафтове маршмелоу ручної роботи
- для гарячого какао
- для подарункових наборів
- для десертної подачі
- незвичні смаки / авторські смаки

---

### 6. Оновити JSON-LD / schema, якщо там є `gourmet marshmallow`
Перевір schema на сторінці `/marshmallow`:
- CollectionPage
- BreadcrumbList
- ItemList
- FAQPage
- інші релевантні JSON-LD блоки

Якщо в назвах/описах schema є акцент на `gourmet marshmallow`, заміни на українське позиціонування:
- `крафтове маршмелоу`
- `маршмелоу ручної роботи`
- `маршмелоу для какао`

Не ламай структуру schema.
Не додавай фейкових даних.
Усе має відповідати видимому контенту сторінки.

---

### 7. Оновити Open Graph
Перевір:
- `og:title`
- `og:description`

Вони мають бути синхронізовані з новим позиціонуванням сторінки.

---

### 8. Перевір canonical
Canonical не чіпай, якщо він уже правильний.

Має залишитися:
`https://chocopix.store/marshmallow`

---

### 9. Не чіпати без потреби
Не змінюй без необхідності:
- route
- slug
- sitemap
- prerender config
- robots
- інші сторінки
- глобальні SEO helper-и, якщо достатньо локальної правки сторінки

---

## Які файли найімовірніше треба перевірити
Насамперед перевір і прав, де потрібно:
- `frontend/src/pages/MarshmallowPage.tsx`

Якщо є helper-и або shared SEO config, що прямо впливають на цю сторінку — акуратно онови лише те, що треба саме для `/marshmallow`.

---

## Що не треба робити
- не додавати англомовне `gourmet marshmallow` як головний ключ
- не писати довгий SEO-блок на пів сторінки
- не спамити словами `маршмелоу купити`
- не робити редизайн
- не чіпати інші категорії
- не робити “покращення про всяк випадок”

---
