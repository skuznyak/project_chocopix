# Шаблон для додавання нового товару

Цей файл допоможе швидко додати новий товар у каталог.

---

## 📝 Шаблон товару (скопіюйте та заповніть)

```typescript
{
  id: 'unique-slug',                    // Унікальний ідентифікатор (латиниця, дефіси)
  slug: 'unique-slug',                  // Такий самий для URL
  name: 'Назва товару',                 // Наприклад: "Карамельна хмаринка"
  shortDescription: 'Короткий опис',    // 1 рядок: "Молочний шоколад, карамель і маршмелоу."
  description: `Повний опис товару.

Термін придатності: 2 місяці. Зберігати в оригінальному пакуванні в темному сухому місці при температурі до +25°C.`,
  composition: ['інгредієнт 1', 'інгредієнт 2'],  // Склад
  weight: 90,                           // Вага в грамах
  flavor: 'Смак',                       // Наприклад: "Карамель", "Полуниця"
  price: 85,                            // Ціна в грн
  // previousPrice: 100,                // Стара ціна (якщо є знижка, розкоментуйте)
  // badge: 'hit',                      // 'hit' | 'new' | 'sale' (опціонально)
  category: 'classic',                  // 'classic' | 'premium' | 'gift-set' | 'cups'
  tags: ['тег1', 'тег2', 'бомбочки'],   // Обов'язково додайте: 'бомбочки', 'набори' або 'чашки'
  popularity: 90,                       // 1-100 (чим більше, тим вище в сортуванні)
  images: [
    { id: '1', src: '/products/slug/1.jpg', alt: 'Фото 1' },
    { id: '2', src: '/products/slug/2.jpg', alt: 'Фото 2' },
  ],
  // featured: true,                    // Розкоментуйте для рекомендованих товарів
},
```

---

## 📋 Поля докладно

### Обов'язкові поля

| Поле | Тип | Приклад | Опис |
|------|-----|---------|------|
| `id` | string | `'caramel-cloud'` | Унікальний ID, латиниця, дефіси замість пробілів |
| `slug` | string | `'caramel-cloud'` | Зазвичай такий самий як `id`, для URL товару |
| `name` | string | `'Карамельна хмаринка'` | Назва товару |
| `shortDescription` | string | `'Молочний шоколад, карамель...'` | Короткий опис (1 рядок) |
| `description` | string | `'Шоколадна бомбочка...'` | Повний опис + термін зберігання |
| `composition` | string[] | `['шоколад', 'маршмелоу']` | Масив інгредієнтів |
| `weight` | number | `90` | Вага в грамах |
| `flavor` | string | `'Карамель'` | Основний смак |
| `price` | number | `85` | Ціна в гривнях |
| `category` | string | `'classic'` | Див. варіанти нижче |
| `tags` | string[] | `['карамель', 'бомбочки']` | Теги для фільтрації |
| `popularity` | number | `95` | Популярність (1-100) |
| `images` | array | `[{id, src, alt}]` | Масив зображень |

### Опціональні поля

| Поле | Тип | Приклад | Опис |
|------|-----|---------|------|
| `previousPrice` | number | `104` | Стара ціна (для знижок) |
| `badge` | string | `'sale'` | Бейдж: `'hit'`, `'new'`, `'sale'` |
| `featured` | boolean | `true` | Чи показувати в рекомендованих |

---

## 🏷️ Варіанти категорій

```typescript
category: 'classic'      // Класичні бомбочки
category: 'premium'      // Преміум бомбочки
category: 'gift-set'     // Подарункові набори
category: 'cups'         // Чашки
```

---

## 🏷️ Варіанти бейджів

```typescript
badge: 'hit'   // Хіт продажу (помаранчевий)
badge: 'new'   // Новинка (зелений)
badge: 'sale'  // Знижка (червоний)
```

## 📷 Зображення

### Варіант 1: Локальні файли (рекомендовано)
```typescript

  { id: '1', src: '/products/caramel-cloud/1.jpg', alt: 'Карамельна бомбочка' },
```

## ✅ Чек-лист перед додаванням

- [ ] `id` та `slug` унікальні (немає дублікатів у файлі)
- [ ] `tags` містить потрібну категорію: `'бомбочки'`, `'набори'` або `'чашки'`
- [ ] `category` відповідає типу товару
- [ ] Додано термін зберігання в `description`
- [ ] `images` містить хоча б 1 зображення
- [ ] `popularity` встановлено (впливає на порядок показу)
- [ ] Перевірено на помилки TypeScript (`npm run build`)

---

## 🚀 Як додати товар

1. Відкрийте `/shared/catalog/products.ts`
2. Знайдіть потрібну секцію (`БОМБОЧКИ`, `НАБОРИ`, `ЧАШКИ`)
3. Скопіюйте шаблон вище
4. Вставте перед закриваючою `]` масиву
5. Заповніть всі поля
6. Збережіть та перевірте: `npm run build`
